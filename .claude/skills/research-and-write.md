# Research and Write Skill

When asked to research a topic and create a note:

1. **Search existing notes**: Use `search_notes` to find what the user already has on this topic
2. **Create note**: Use `notes_create` with a clear title and structured markdown:
   - Start with a brief overview paragraph
   - Use `##` sections for major topics
   - Use bullet lists for key points
   - Include a `## Sources` section at the end
3. **Cross-reference**: If related notes exist, mention them with "See also: [Note Title]"
4. **Polish**: Review the note structure — ensure headings flow logically

## Note Structure Template

```markdown
# [Topic]

Brief overview paragraph.

## Key Points

- Point 1
- Point 2

## Details

Expanded discussion...

## Sources

- Source 1
- Source 2
```
