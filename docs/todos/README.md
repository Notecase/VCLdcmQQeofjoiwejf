# Plan Archives

This folder contains archived Claude Code plan-mode plans for the Inkdown project.

## Naming Convention

Files follow the pattern: `YYYY-MM-DD-<short-kebab-case-topic>.md`

Examples:

- `2026-02-06-add-plan-archive.md`
- `2026-02-06-secretary-roadmap-daily-sync.md`

## Purpose

Every plan-mode plan is saved here automatically so there's a browsable history of all implementation plans, decisions, and their rationale.
